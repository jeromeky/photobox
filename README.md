Photobox

========

1) Install dajaxice

2) Install solr-thumbnail

3) Install PIL

4) Install memcached (memcached.org) (code.google.com/memcached/wiki)
	./configure
	make && make test
	sudo make install
	
5) Install client for memcached (ftp://ftp.tummy.com/pub/python-memcached)
	python setup.py install

6) Install photobox
